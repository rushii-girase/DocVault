# DocVault 🔐📚

**An Institutional Document Management System for seamless document handling, verification, and archival in educational institutions.**

A comprehensive full-stack solution designed to streamline document management across students, staff, and administrators with secure uploads, role-based access, and audit logging.

---

## 📋 Overview

DocVault is a robust institutional platform that centralizes document management for educational institutions. It provides a secure, organized system for:

- **Students** – Upload required documents, track submission status, and manage their academic records
- **Staff/Faculty** – Review, verify, and process student documents with detailed remarks
- **Administrators** – Manage staff accounts, monitor audit logs, and oversee institutional document workflows

The platform ensures document integrity through comprehensive audit trails, role-based access control, and a modern, intuitive user interface.

---

## 🎯 Key Features

### 👥 Role-Based Access Control
- **Admin Dashboard** – Full institutional oversight, staff management, audit logs
- **Staff Portal** – Document review, verification workflow, batch document requests
- **Student Portal** – Document upload, status tracking, profile management

### 📄 Document Management
- **Secure Upload** – File handling with size validation
- **Document Preview** – In-browser document preview without downloads
- **Download Support** – Verified documents available for download
- **Status Tracking** – Real-time document status (Pending, Approved, Rejected)
- **Document Remarks** – Staff can add verification remarks to documents

### 🔐 Security & Compliance
- **Email Verification** – Two-step email verification during registration
- **Audit Logging** – Complete audit trail of all document operations
- **Staff Blocking** – Admin can block/unblock staff accounts
- **Secure Authentication** – JWT-based authentication system
- **Password Encryption** – Bcrypt password hashing

### 📊 Required Documents Management
- **Dynamic Document Types** – Define required documents by college and course
- **Completion Tracking** – Visual tracking of submitted vs. required documents
- **Batch Requests** – Staff can request missing documents from all students

### 🎨 Modern User Interface
- **Material Design System** – Consistent Material Design 3 styling
- **Responsive Design** – Works seamlessly on desktop, tablet, and mobile
- **Real-Time Updates** – Instant status updates and notifications
- **Tailwind CSS** – Modern, utility-first styling approach
